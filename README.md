# Segurança no<br /> Trabalho
<strong><p>Ambientes empresariais, industriais e operacionais frequentemente lidam com riscos invisíveis relacionados à presença de gases tóxicos, inflamáveis ou potencialmente explosivos. Vazamentos de gás podem causar:</p></strong>

<li>Intoxicação de trabalhadores</li>

<li>Incêndios ou explosões</li>

<li>Contaminação interna em prédios corporativos</li>

<li>Falta de informação em tempo real para equipes de segurança</li>

<li>Reação lenta em situações críticas</li>

#

>> E Para solucionarmos??
<p>Desenvolvemos um protótipo de monitoramento inteligente de gases, nossa solução consegue:</p>

<li>✔ Detectar gases tóxicos e inflamáveis</li>

<li>✔ Calcula PPM (partes por milhão)</li>

<li>✔ Classifica o ambiente como NORMAL / AVISO / CRÍTICO</li>

<li>✔ Envia tudo automaticamente via MQTT</li>

<li>✔ Permite integração com dashboards, apps e sistemas prediais</li>

<li>✔ Funciona em indústrias, galpões e prédios corporativos</li>
